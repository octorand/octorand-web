Create new lottery game

Anyone can create a lottery game using our game engine. As the game creator, you will earn a share of the prize pool. When creating the game, you get to choose the number of winners and their prize percentages. Games are setup with native Algorand coin ALGO, as the currency. At the end of the game, we will help you choose the winners based on their purchased ticket amounts.
